## 1. 设计概述

### 1.1 产品定位

移动端是助手-贾维斯的**远程监控和管理终端**，主要职责：

- 📊 **实时监控**: 随时随地查看PC端运行状态
- 🎮 **远程控制**: 一键启停任务、Agent和工作流
- 🔔 **即时通知**: 关键事件第一时间推送提醒
- 📈 **数据洞察**: 查看统计报表和性能分析
- ⚙️ **便捷管理**: 移动化的配置和管理体验

**核心原则**: 移动端不执行复杂自动化任务，所有执行在PC端完成，移动端专注于监控、管理和控制。

### 1.2 设计理念

**触手可及 (Accessible)**
- 重要信息一目了然
- 常用操作快速触达
- 紧急控制即时响应

**实时掌控 (Real-time)**
- WebSocket实时数据推送
- 执行状态动态更新
- 通知及时准确送达

**智能高效 (Efficient)**
- 智能化的信息聚合
- 简化的操作流程
- 手势驱动的交互

### 1.3 设计目标

- ✅ 适配iOS和Android双平台
- ✅ 支持4.7寸至12.9寸屏幕
- ✅ 暗黑模式完整支持
- ✅ 流畅的60fps动画
- ✅ 低于100MB内存占用
- ✅ 离线也能查看历史数据

## 3. 导航架构

### 3.1 底部Tab导航

采用**5个Tab**的标准移动端导航模式：

```
┌────────────────────────────────────────┐
│                                        │
│          [主要内容区域]                │
│                                        │
└────────────────────────────────────────┘
┌────────────────────────────────────────┐
│  🏠      📊      📞      🔔      👤    │
│ 首页    任务    通话    通知    我的   │
└────────────────────────────────────────┘
```

**Tab详细说明**:

| Tab | 图标 | 名称 | 功能 | 徽章 |
|-----|------|------|------|------|
| 1 | 🏠 | 首页 | 仪表盘、快速操作、统计概览 | - |
| 2 | 📊 | 任务 | 任务列表、工作流管理、执行监控 | 运行中数量 |
| 3 | 📞 | 通话 | AI通话记录、通话详情 | 新通话 |
| 4 | 🔔 | 通知 | 通知中心、系统消息 | 未读数量 |
| 5 | 👤 | 我的 | 个人设置、账号管理 | - |

**Tab状态**:
- **未选中**: 灰色图标 (#8C8C8C) + 灰色文字
- **选中**: 金色图标 (#D4AF37) + 金色文字
- **有徽章**: 红色圆点或数字徽章

### 3.2 页面层级

```
Level 1: 底部Tab页面 (5个主页面)
    ↓
Level 2: 列表/详情页 (下钻页面)
    ↓
Level 3: 详细信息/编辑页 (更深层级)
    ↓
Level 4: 全屏模态页 (如图片预览)
```

**导航模式**:
- **Push导航**: 标准的页面跳转，带返回按钮
- **Modal导航**: 弹出式页面，带关闭按钮
- **Tab切换**: 平行页面切换，保持状态

### 3.3 手势导航

```
← 右滑: 返回上一页 (iOS标准)
↓ 下拉: 刷新当前页面
↑ 上滑: 加载更多数据
长按:   显示上下文菜单
```
