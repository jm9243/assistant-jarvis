## 1. 设计概述

### 1.1 产品定位

管理后台是**企业级B端管理系统**，服务于以下角色：

- 🔐 **超级管理员**: 拥有所有权限，负责系统整体管理
- 👥 **运营管理员**: 负责用户管理、内容审核、数据运营
- 💰 **财务管理员**: 负责订单、支付、发票等财务数据
- ✅ **审核管理员**: 负责模板审核、内容审核
- 📞 **客服人员**: 查看用户信息、处理工单

### 1.2 设计理念

**专业 (Professional)**
- 专业的企业级设计语言
- 严谨的数据展示
- 清晰的信息架构

**高效 (Efficient)**
- 批量操作支持
- 快捷键支持
- 智能筛选和搜索

**数据驱动 (Data-Driven)**
- 丰富的数据可视化
- 实时数据监控
- 多维度数据分析

### 1.3 设计目标

- ✅ 基于Ant Design Pro设计规范
- ✅ 支持暗黑模式
- ✅ 响应式设计（1280px-2560px）
- ✅ 数据可视化丰富
- ✅ 操作效率优先
- ✅ 完善的权限控制

### 5.2 关键指标卡片

**设计要点**
- 4列网格布局（响应式：大屏4列，中屏2列，小屏1列）
- 图标 + 数值 + 标题 + 变化趋势
- 数值大而突出（32px Bold）
- 趋势箭头颜色：上升绿色、下降红色、持平灰色
- 悬停显示详细数据
- 支持点击跳转到详细页面

**数据项**
1. **总用户数**: 平台总注册用户数
2. **本月收入**: 当月总收入金额
3. **任务执行**: 本月工作流执行总次数
4. **活跃用户**: DAU（日活跃用户）

```

**小屏 (<1280px) - 最小支持**
```
