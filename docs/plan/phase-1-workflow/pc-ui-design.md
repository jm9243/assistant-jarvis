## 设计概述

### 设计原则
- **极简至上**: 复杂功能简单化
- **对话优先**: 自然语言交互
- **沉浸式未来感**: 钢铁侠贾维斯的科技美学
- **隐形设计**: 功能强大但UI克制

### 色彩系统
- 主色: Jarvis Gold (#FFB800)
- 背景: Deep Space (#0A0E27)
- 卡片: Dark Panel (#1A1F3A)
- 文字: 白色 (#FFFFFF) / 灰色 (#A8B2D1)

### 字体系统
- 英文: Orbitron (标题) / Rajdhani (正文)
- 中文: HarmonyOS Sans SC / PingFang SC
- 代码: JetBrains Mono
