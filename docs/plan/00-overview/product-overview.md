# 助手-贾维斯 产品概述

**版本**: V2.0  
**日期**: 2025-11-08

---

## 1. 产品愿景

"助手-贾维斯"是一款跨平台智能助理，以强大的RPA为执行基础，以先进的Multi-Agent系统为决策核心，致力于将用户从重复、繁琐的数字化工作中解放出来。

**核心设计哲学：驾驶舱与引擎分离**
- **驾驶舱（Frontend）**：Tauri 2 + React构建的轻量UI，负责交互和可视化
- **引擎（Backend）**：独立的Python Sidecar后台进程，执行所有自动化逻辑

---

## 2. 核心价值

- **极致自动化**: 智能录制器 + 可视化工作流设计器
- **AI驱动**: Multi-Agent系统，自然语言交互
- **跨平台协同**: PC执行、移动端监控、云端管理
- **开放生态**: 模板市场、工具商店

---

## 3. 目标用户

### 高效能办公族
- **痛点**: 重复工作占据大量时间
- **需求**: 自动化固定流程
- **关注**: 工作流录制、数据提取、报表生成

### 技术爱好者
- **痛点**: 需要更强大的自动化能力
- **需求**: 可定制的复杂工作流
- **关注**: 高级编排、API集成、Agent开发

### 小微企业主
- **痛点**: 人手有限，无法休息
- **需求**: 7×24小时的"数字员工"
- **关注**: AI通话接听、智能客服、订单处理

### 企业IT管理员
- **痛点**: 需要统一的自动化解决方案
- **需求**: 企业级管理后台
- **关注**: 用户管理、权限控制、数据审计

## 6. 技术架构

### PC端
- **前端**: Tauri 2 + React + TypeScript
- **引擎**: Python Sidecar (FastAPI)
- **GUI自动化**: pywinauto (Windows) / pyobjc (macOS)
- **Web自动化**: Playwright
- **本地存储**: SQLite

### 云端服务
- **后端**: Go (Gin/Fiber)
- **数据库**: Supabase (PostgreSQL)
- **认证**: Supabase Auth
- **存储**: Supabase Storage
- **实时**: Supabase Realtime

### 移动端
- **框架**: React Native
- **状态管理**: Zustand
- **UI组件**: React Native Paper

## 10. 风险与挑战

### 技术风险
- 跨平台GUI自动化兼容性
- 虚拟音频设备稳定性
- Multi-Agent协同复杂度

### 市场风险
- 用户教育成本
- 竞品压力
- 商业模式验证

### 应对策略
- 提前技术验证（POC）
- 分阶段迭代降低风险
- 持续用户反馈优化
