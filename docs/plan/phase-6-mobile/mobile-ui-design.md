# Phase 6: 移动端 - UI设计

**版本**: V2.0  
**日期**: 2025-11-08  
**设计理念**: 触手可及·实时掌控·智能高效

---

## 设计概述

### 设计原则
- **触手可及**: 重要信息一目了然
- **实时掌控**: WebSocket实时数据推送
- **智能高效**: 简化的操作流程

### 色彩系统
- 主色: #D4AF37 (贾维斯金)
- 背景: #0A0E14 (深色) / #FFFFFF (浅色)
- 状态色: 在线#52C41A, 离线#8C8C8C, 运行中#1890FF

### 适配规范
- 小屏: 375x667 (iPhone SE)
- 中屏: 390x844 (iPhone 14)
- 大屏: 428x926 (iPhone 14 Pro Max)

## 组件规范

### 卡片组件

```css
背景: rgba(37, 42, 74, 0.8) (暗色) / #FFFFFF (浅色)
边框: 1px solid rgba(255,255,255,0.05)
圆角: 12pt
Padding: 16pt
阴影: 0 2pt 8pt rgba(0,0,0,0.1)

悬停:
  transform: scale(1.02)
```

### 按钮组件

```css
Primary:
  背景: #D4AF37
  文字: #FFFFFF
  高度: 44pt
  圆角: 8pt

Secondary:
  背景: 透明
  边框: 1px solid #D4AF37
  文字: #D4AF37
```

### 列表项组件

```css
高度: 60pt
背景: 透明
边框底部: 1px solid rgba(255,255,255,0.1)
Padding: 12pt 16pt

点击:
  背景: rgba(255,184,0,0.1)
```

## 手势交互

### 支持的手势

```
← 右滑: 返回上一页 (iOS标准)
↓ 下拉: 刷新当前页面
↑ 上滑: 加载更多数据
长按:   显示上下文菜单
左滑:   快捷操作（删除、暂停等）
```

## 适配规范

### 安全区域

```
顶部状态栏: 动态适配刘海屏
底部Home指示器: iOS 34pt, Android 16pt
左右边距: 16pt (标准内容区)
```

### 响应式布局

```
小屏 (375x667):
  - 卡片单列显示
  - 字体适当缩小
  - 减少间距

中屏 (390x844):
  - 标准布局
  - 正常字体大小

大屏 (428x926):
  - 增加内容宽度
  - 部分区域双列显示
```
