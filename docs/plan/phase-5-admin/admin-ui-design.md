# Phase 5: 管理后台 - UI设计

**版本**: V2.0  
**日期**: 2025-11-08  
**设计理念**: 专业·高效·数据驱动

---

## 设计概述

### 设计原则
- **专业**: 企业级设计语言
- **高效**: 批量操作、快捷键支持
- **数据驱动**: 丰富的数据可视化

### 色彩系统
- 主色: #D4AF37 (贾维斯金-柔和版)
- 功能色: Success #52C41A, Warning #FAAD14, Error #FF4D4F
- 背景: #F5F5F5 (浅色) / #141414 (暗色)

### 布局架构
- 顶部导航 (64px) + 侧边栏 (208px) + 内容区

```css
表头:
  背景: #FAFAFA
  高度: 48px
  字体: 14px Semibold
  边框: 1px solid #F0F0F0

表格行:
  高度: 56px
  字体: 14px Regular
  边框: 1px solid #F0F0F0
  
  悬停:
    背景: #FAFAFA
```

```css
背景: #FFFFFF
边框: 1px solid #F0F0F0

  阴影: 0 4px 12px rgba(0,0,0,0.15)
```
