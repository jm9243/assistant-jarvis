## 1. 设计理念

### 1.1 核心理念

> **"让AI成为你的数字伙伴，而不是一个工具软件"**

**设计原则**：
- **极简至上**: 复杂功能简单化，能自动完成的绝不让用户手动
- **对话优先**: 用自然语言交互，而非点击按钮和填写表单
- **沉浸式未来感**: 钢铁侠贾维斯的科技美学，全息、悬浮、流动
- **隐形设计**: 功能强大但UI克制，不打扰时几乎隐形

### 1.2 交互哲学

**"贾维斯不是应用，是伙伴"**

传统应用的交互模式：
```
用户 → 打开应用 → 点击菜单 → 填写表单 → 点击按钮 → 等待结果
```

贾维斯的交互模式：
```
用户 → 呼叫"贾维斯" → 说出需求 → 自动执行 → 实时反馈
```

**设计目标**：
- ✅ 0学习成本：说话就能用
- ✅ 0视觉干扰：不用时悬浮球静默待命
- ✅ 0复杂操作：AI理解意图，自动执行

## 2. 视觉系统

### 2.1 色彩系统

#### 主色调：贾维斯金（Jarvis Gold）

灵感来源：钢铁侠实验室中的全息投影和反应堆光芒

```
主色系（Primary）:
├─ Jarvis Gold Primary:   #FFB800  // 标志性科技黄
├─ Jarvis Gold Light:     #FFD966  // 高亮、悬停
├─ Jarvis Gold Dark:      #CC9300  // 按下、激活
└─ Jarvis Gold Glow:      #FFB800 + 60% opacity + blur  // 发光效果

辅助色系（Secondary）:
├─ Arc Blue:              #00D9FF  // 反应堆蓝，用于次要信息
├─ Cyber Purple:          #9D4EDD  // AI思考状态
├─ Success Green:         #00F5A0  // 成功状态
├─ Warning Orange:        #FF6B35  // 警告状态
└─ Error Red:             #FF006E  // 错误状态

背景色系（Background）:
├─ Deep Space:            #0A0E27  // 主背景
├─ Dark Panel:            #1A1F3A  // 卡片/面板背景
├─ Elevation 1:           #252A4A  // 悬浮层1
├─ Elevation 2:           #2E3558  // 悬浮层2
└─ Overlay:               #0A0E27 + 80% opacity  // 遮罩层

文字色系（Text）:
├─ Text Primary:          #FFFFFF  // 主要文字
├─ Text Secondary:        #A8B2D1  // 次要文字
├─ Text Tertiary:         #6B7A99  // 三级文字
└─ Text Disabled:         #3D4663  // 禁用文字
```

#### 色彩使用规范

**主色（Jarvis Gold）**：
- ✅ 悬浮球主体颜色
- ✅ 主要操作按钮
- ✅ 重要状态指示
- ✅ 选中/激活状态
- ❌ 不用于大面积填充

**辅助色使用场景**：
- `Arc Blue`: 超链接、次要按钮、信息提示
- `Cyber Purple`: AI正在思考、处理中状态
- `Success Green`: 任务完成、执行成功
- `Warning Orange`: 需要注意的信息
- `Error Red`: 错误、失败、危险操作

## 3. 核心交互模式

### 3.1 悬浮球（Jarvis Orb）- 主交互入口

#### 3.1.1 视觉设计

**位置**：
- 默认位置：屏幕右下角，距离边缘 32px
- 可拖拽到屏幕任意位置
- 自动吸附到屏幕边缘（可配置）

**尺寸**：
- 待命状态：64px × 64px
- 激活状态：72px × 72px（放大动画）
- 展开状态：320px × 480px（对话界面）

**外观设计**：

```
待命状态（Idle）:
┌─────────────────────────┐
│                         │
│      ╱◉╲               │  // 金色圆球
│     ◢███◣              │  // 中心有反应堆纹理
│     ◥███◤              │  // 轻微呼吸光效
│      ╲◉╱               │  // 半透明材质
│                         │
│   "贾维斯 待命中"        │  // Tooltip
└─────────────────────────┘

激活状态（Active）:
┌─────────────────────────┐
│                         │
│      ╱◉╲               │  // 金色圆球放大
│    ◢█████◣             │  // 强烈脉冲光效
│    ████████            │  // 波纹扩散
│    ◥█████◤             │  // 全息光晕
│      ╲◉╱               │
│                         │
│  "正在聆听..."          │  // Tooltip
└─────────────────────────┘

思考状态（Thinking）:
- 紫色光效闪烁
- 内部粒子旋转
- 外圈呼吸动画

执行状态（Working）:
- 金色光效流动
- 进度环绕圆球
- 快速脉冲效果
```

**材质效果**：
- 玻璃质感（Glassmorphism）
- 50% 透明度背景
- 背景模糊（backdrop-filter: blur(20px)）
- 金色渐变边框（2px，发光）
- 动态光晕（box-shadow动画）

#### 3.1.2 交互逻辑

**唤醒方式**：
1. **点击悬浮球** → 展开对话界面
2. **语音唤醒** → "嘿，贾维斯" / "Jarvis" → 语音输入模式
3. **全局快捷键** → `Cmd/Ctrl + Space` → 展开对话界面
4. **双击桌面空白** → 唤醒（可配置）

**状态流转**：
```
待命 → 点击/唤醒 → 激活 → 聆听 → 思考 → 执行 → 完成 → 待命
  ↑                                                    ↓
  └────────────────── 取消/关闭 ←──────────────────────┘
```

**悬浮球菜单**：
- 右键点击悬浮球 → 快捷菜单
  - 📊 打开主界面
  - ⚡ 最近使用的工作流（Top 5）
  - 🎙️ 开启/关闭AI接听
  - 🔇 静音模式
  - ⚙️ 设置
  - 📤 退出

### 4.4 工作流设计器（Studio）

#### 4.4.1 屏幕结构

```
┌────────────── 顶部工具栏 (56px) ──────────────┐
│  ← 工作流列表   |  工作流名称输入框           │  [保存] [发布]
│  状态指示 🟢/🟡/🔴   |  版本号  |  最近保存时间     │
└───────────────────────────────────────────┘
┌── 左侧面板 280px ──┬────── 画布区域 (自适应) ─────┬── 右侧面板 320px ──┐
│ 节点库 Tab         │ ┌──────── 画布 ────────┐ │ │ 属性/配置 Tab     │
│ • UI自动化         │ │  React Flow无限画布 │ │ │ • 基础信息         │
│ • 控制流           │ │  (支持缩放/平移)    │ │ │ • 元素定位         │
│ • 集成             │ └────────────────────┘ │ │ • 参数配置         │
│ • 系统/文件        │    ▽ 小地图 (右下角  )    │ │ • 错误处理         │
│ • AI/通知/Agent    │                          │ │ • 执行策略         │
└────────────────────┴─────────────────────────┴──────────────────────┘
```

#### 4.4.2 节点库交互
- 左侧面板支持折叠分组，搜索框可实时过滤节点名称/标签。
- 拖拽节点到画布即可创建实例，悬浮提示展示描述与必填参数。
- 支持将常用节点收藏至“Favorites”，置顶显示；空状态引导“拖拽创建第一个节点”。

#### 4.4.3 画布与连线
- 画布默认显示网格+吸附，`Cmd/Ctrl + 鼠标滚轮` 缩放，空格+拖动为平移。
- 提供对齐线、节点分布（快捷键 `Shift + A/D`），框选（`Shift + 拖拽`）。
- 连接点悬停展示输入/输出类型；条件分支显示标签（True/False、自定义文本）。
- 顶部工具栏包含撤销/重做、缩放重置、自动布局、Minimap显隐、导入/导出JSON。

#### 4.4.4 节点配置面板
- 右侧面板使用分节折叠：
  - **基础信息**：名称、图标、颜色、描述；支持实时预览。
  - **元素定位**：策略选择器（AXUI/OCR/Image/Position），每项可展开填写属性、超时、重试次数。
  - **操作参数**：根据节点类型动态渲染表单（如 Click: 点击类型、修饰键、偏移量）。
  - **错误处理**：失败策略（终止/继续/跳转节点），可附带重试次数与间隔。
  - **条件执行**：启用开关后出现变量表达式编辑器，支持语法高亮与校验。
- 表单保存采用自动草稿，顶部显示“已保存/保存中”状态和最近保存时间；未保存时关闭弹出确认对话框。

#### 4.4.5 节点列表与版本
- 画布上方显示版本标签（如 v1.2），点击打开版本侧栏：版本对比（Diff）、回滚按钮、更新日志输入框。
- “兼容性提示”徽章在节点标题旁展示，如检测到目标应用缺失会出现橙色警告，可点击查看详情。

### 4.5 智能录制器（Recorder）

#### 4.5.1 启动浮层
- 点击“🎙️ 录制工作流”或快捷键 `Cmd/Ctrl + Shift + R` 弹出圆角浮层：
  - Stepper：选择录制模式（全自动/手动）、目标应用、保存位置。
  - 权限检查提示（屏幕录制、辅助功能、麦克风）。
- “开始录制”后最小化为悬浮控件，停靠屏幕边缘。

#### 4.5.2 悬浮控件
```
┌─────── Recorder Mini ───────┐
│ ⏺ 正在录制 | 00:02:35        │
│ [暂停] [继续] [停止] [标记]   │
│ 当前操作：Click - 企业微信发送按钮 │
└──────────────────────────┘
```
- “标记”可插入注释，结束录制后在工作流中以注释节点呈现。
- 暂停时控件变灰，并提示“录制已暂停”。

#### 4.5.3 叠加窗口状态
- **Hover高亮**：黄色描边 + 半透明遮罩，顶部显示元素名称/路径。
- **选中**：蓝色实线 + 节点编号，浮动面板显示将生成的节点类型与主要参数，允许即时修改。
- **多选**：手动模式下可 `Shift + 点击` 依次捕获。
- 屏幕右下角显示录制步骤列表（迷你时间线），支持撤销上一步。

#### 4.5.4 结果预览弹窗
- 停止录制后弹出总结：录制总步数、识别到的应用、定位成功率。
- 列表展示每个生成节点，可在此重命名、删除或合并相邻步骤；确认后生成工作流并跳转设计器。

## 6. 动效设计

### 6.1 微动效原则

**设计理念**：动效应该"感觉对了"而不是"炫技"

**原则**：
- ✅ 自然流畅（模拟物理世界）
- ✅ 快速响应（< 300ms）
- ✅ 有意义（每个动效都有目的）
- ✅ 可关闭（性能模式可禁用动效）

### 6.2 核心动效

#### 6.2.1 悬浮球动效

**呼吸光效（Idle）**：
```javascript

持续时间：2s
缓动函数：ease-in-out
循环：infinite

关键帧：
0%:   scale(1.0), opacity(0.8)
50%:  scale(1.05), opacity(1.0)
100%: scale(1.0), opacity(0.8)
```

**激活脉冲（Active）**：
```javascript
动画：波纹扩散
持续时间：0.6s
缓动函数：cubic-bezier(0.4, 0, 0.2, 1)

波纹：
- 从中心向外扩散
- 金色，透明度从0.8衰减到0
- 3层波纹依次触发
```

**思考动画（Thinking）**：
```javascript
动画：粒子旋转
持续时间：1.5s
缓动函数：linear
循环：infinite

效果：
- 内部6个小粒子围绕中心旋转
- 紫色光效
- 旋转速度逐渐加快
```

const drawWaveform = (
  canvas: HTMLCanvasElement,
  audioData: Uint8Array
) => {
  const ctx = canvas.getContext('2d')!;
  const width = canvas.width;
  const height = canvas.height;
  const barWidth = 4;
  const gap = 2;
  const barCount = Math.floor(width / (barWidth + gap));
  
  ctx.clearRect(0, 0, width, height);
  
  // 金色渐变
  const gradient = ctx.createLinearGradient(0, 0, 0, height);
  gradient.addColorStop(0, '#FFD966');
  gradient.addColorStop(1, '#FFB800');
  ctx.fillStyle = gradient;
  
  for (let i = 0; i < barCount; i++) {
    const dataIndex = Math.floor(i * (audioData.length / barCount));
    const barHeight = (audioData[dataIndex] / 255) * height;
    const x = i * (barWidth + gap);
    const y = (height - barHeight) / 2;
    
    ctx.fillRect(x, y, barWidth, barHeight);
  }
};

```typescript
const LazyImage = ({ src, alt, ...props }) => {
  const [imageSrc, setImageSrc] = useState<string>();
  const imageRef = useRef<HTMLImageElement>(null);
  
  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setImageSrc(src);
          observer.disconnect();
        }
      },
      { threshold: 0.1 }
    );
    
    if (imageRef.current) {
      observer.observe(imageRef.current);
    }
    
    return () => observer.disconnect();
  }, [src]);
  
  return (
    <img
      ref={imageRef}
      src={imageSrc || '/placeholder.png'}
      alt={alt}
      {...props}
    />
  );
};
```
